# SysY2022：基于flex的词法分析(C语言实现)

在文件sysy.l中补充适当的此法规则和动作代码，设计识别`SysY2022`语言单词符号的词法分析器。

你需要补充的内容有：<p>
(1) 标识符ID;<p>
(2) int型字面量INT_LIT;<p>
(3) float型字面量FLOAT_LIT;<p>
(4) 词法错误LEX_ERR<p>

等此法规则的定义,以及对应的动作代码。<p>
